# Schleuse-Skeleton (Kopiervorlage)

Beim Anlegen einer neuen Schleuse `schleusen/<name>/`: Inhalt dieses Verzeichnisses kopieren, `<name>` ersetzen, Tool-Module schreiben.

Wird mit der ersten konkreten Schleuse (`wetter/`) als Referenz-Implementierung befüllt — bis dahin nur Doku.

Spec mit konkretem Code-Skeleton: [`../../docs/schleuse-skeleton.md`](../../docs/schleuse-skeleton.md).
